## Project Overview

ThreatInk is a machine learning-powered spam message detection system that classifies email and SMS messages as **spam** or **ham (legitimate)**. The system uses three machine learning approaches trained on a combined dataset of 94,748 messages from three sources.

### Machine Learning Models

| Approach | Models | Output |
|----------|--------|--------|
| Classification | Naive Bayes, Logistic Regression, Linear SVM, Random Forest | SPAM / HAM + confidence score |
| Regression | Ridge Regression, Gradient Boosting | Spam risk score (0.0 – 1.0) |
| Clustering | K-Means (K=2) | Spam pattern grouping |

### Model Performance Results

| Model | Accuracy | Precision | Recall | F1 Score | ROC-AUC |
|-------|----------|-----------|--------|----------|---------|
| Naive Bayes | 0.9341 | 0.9171 | 0.9506 | 0.9335 | 0.9868 |
| Logistic Regression | 0.9651 | 0.9601 | 0.9685 | 0.9643 | 0.9950 |
| Linear SVM | 0.9691 | 0.9682 | 0.9683 | 0.9683 | 0.9956 |
| **Random Forest ★** | **0.9746** | **0.9696** | **0.9785** | **0.9740** | **0.9965** |

| Regression Model | MAE | RMSE | R² |
|-----------------|-----|------|-----|
| Ridge Regression | 0.0701 | 0.0716 | 0.2718 |
| Gradient Boosting | 0.0464 | 0.0566 | 0.5448 |

| Clustering Model | K | Silhouette Score |
|-----------------|---|-----------------|
| K-Means | 2 | 0.2597 |

## Making Predictions

After running the notebook, use the `predict_spam()` function from Section 9:

```python
# Test a spam message
result = predict_spam("Congratulations! You won $1,000,000! Click here NOW!")
print(result)

# Test a legitimate message
result = predict_spam("Hey, are you free for lunch tomorrow at 12pm?")

# Output: {'label': 'HAM', 'confidence': 0.9965, 'risk_score': 0.03}
```
